# App mobile 
***
* Ứng dụng app mobile viết bằng react-native

## Installation
***
Clone project or download zip

``` slim
 git clone https://github.com/espressif/arduino-esp32.git
```
After downloading, you must run command in root project directory
``` slim
 npm install
```
or
``` slim
yarn install
```
